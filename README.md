# Europol Awareness & Reporting

This is a retro 90s-style website designed to raise awareness about online child exploitation and provide safe resources for reporting illegal content. The site uses bright colours, marquee text, 3D buttons, and CSS animations, while remaining fully safe and functional.

## Features
- Retro 90s design with gradients, marquee, and blinking buttons
- Responsive layout for desktop and mobile
- Links to official organisations: Europol, IWF (UK), Safer Internet Centres, INHOPE
- SEO-friendly with Open Graph, Twitter Cards, and structured data
- Informational disclaimer included

## Usage
Open `index.html` in your browser or host the site on GitHub Pages to make it publicly accessible.
